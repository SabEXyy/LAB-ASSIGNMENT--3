# LAB-ASSIGNMENT--3
Overview
A sleek, responsive personal portfolio website showcasing my skills as a Full Stack Developer. Built with pure HTML, CSS, and vanilla JavaScript, this site features a modern glassmorphism design with smooth scroll effects, gradient backgrounds, and hover animations. It's optimized for both desktop and mobile, perfect for developers looking to present their work professionally.
This project was created as part of a lab assignment (LAB-ASSIGNMENT-3) to demonstrate front-end development proficiency.
Features

Hero Section: Eye-catching introduction with highlighted text and a prominent "Hire Me" CTA.
About Me: Brief bio highlighting passion for clean code and innovative UIs.
Skills Grid: Interactive cards displaying key technologies like HTML5, CSS3, JavaScript, React, Node.js, MongoDB, Git & GitHub, and Express.js.
Projects Showcase: Grid of featured projects including E-Commerce Website, Netflix Clone, Real-Time Chat App, Weather Dashboard, Task Manager, and Blog Platform.
Contact Form: Simple, styled form for inquiries (fields: Name, Email, Message).
Responsive Design: Mobile-first approach with media queries for seamless viewing on all devices.
Smooth Animations: Header scroll effects, card hovers, and button transitions for enhanced UX.

Tech Stack

HTML5 for semantic structure
CSS3 with Flexbox/Grid, gradients, and backdrop filters
JavaScript for scroll-based interactions
Fonts: Poppins (body) & Orbitron (headings)
Icons: Font Awesome 6.6.0
